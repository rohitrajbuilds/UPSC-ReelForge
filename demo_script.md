# Demo Script

## 2-Minute Walkthrough

1. Introduce the problem: UPSC content teams spend too much time manually converting syllabus topics into short vertical videos.
2. Open UPSC ReelForge and show the Generate Video page.
3. Enter a topic such as `Directive Principles of State Policy`, keep the category as `Polity`, and click `Generate Video`.
4. Show the live progress tracker moving through script, voice, animation, subtitles, and video assembly.
5. Open the final preview area and point out the download button for the generated MP4.
6. Move to the Daily Agent page and explain the videos-per-day setting, generated-today count, last run, and next run.
7. Trigger `Run Agent Once` to demonstrate autonomous topic queue processing.
8. Open Output History and show how each generated video is logged for auditability.
9. Close by noting that the current project uses original placeholder content and local generation, while future versions could add optional LLM assistance and ElevenLabs integration.
